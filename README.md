📝 Flask Task Manager App

A simple Task Management Web Application built using Flask, SQLAlchemy, and SQLite.
Users can register, log in, create tasks, mark them as completed, and manage their daily schedule.

🚀 Features

🔐 User Registration & Login (Password Hashing)

➕ Add Tasks with Start & End Date

✅ Mark Tasks as Completed

🗑 Delete Individual Tasks

🧹 Clear All Tasks

📅 Automatic Task Status:

Upcoming

Ongoing

Completed

Expired

👤 Session-based Authentication

🛠 Tech Stack

Backend: Flask

Database: SQLite

ORM: Flask-SQLAlchemy

Authentication: Werkzeug Security

Frontend: HTML, CSS
